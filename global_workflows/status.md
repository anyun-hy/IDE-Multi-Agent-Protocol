# /status 全局状态

## 1. 查看项目大盘
// turbo
调用 `get_project_status()` 获取所有活跃项目的当前阶段和待办事项。

## 2. 查看最近决策和记忆
// turbo
调用 `recall_memories(limit=10)` 获取最近的全局记忆。

## 3. 查看可恢复的角色快照
依次尝试调用 `load_context()` 检查以下角色是否有存档：
- Researcher
- Writer
- Engineer
- Office
