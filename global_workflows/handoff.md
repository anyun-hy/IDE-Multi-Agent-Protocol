# /handoff 存盘交接

## 1. 总结当前状态
整理以下信息为一段摘要：
- 当前角色名称
- 正在执行的任务
- 已完成的内容要点
- 未完成的内容
- 最近做出的关键决策

## 2. 保存角色快照
// turbo
调用 `save_context(角色名, 当前任务, 摘要文本)` 将状态写入 MCP 数据库。

## 3. 更新项目状态（如适用）
// turbo
如果当前工作关联了某个项目，调用 `update_project_status(项目名, 阶段, 下一步待办)` 更新大盘。

## 4. 生成交接令牌
输出以下内容供用户在新对话中使用：

```
✅ 存盘完成。请关闭当前对话，新开对话后粘贴以下内容：

[Agent 握手令牌]
@role: {当前角色}
@project: {项目名，如适用}
@task: {未完成的任务}
@context: 请调用 load_context('{角色名}') 恢复上下文
```
